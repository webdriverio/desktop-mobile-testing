## Default Permission

Allows all WebDriverIO plugin commands for testing

#### This default permission set includes the following:

- `allow-execute`
- `allow-set-mock`
- `allow-get-mock`
- `allow-clear-mocks`
- `allow-reset-mocks`
- `allow-restore-mocks`
